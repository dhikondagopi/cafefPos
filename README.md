# CafeFlow POS

CafeFlow POS is a MERN-based restaurant point-of-sale system with admin dashboard, cashier POS terminal, table management, payment flow, customer management, and real-time kitchen display using Socket.IO.

## Tech Stack

- MongoDB + Mongoose
- Express.js
- React + Vite
- Node.js
- Socket.IO
- JWT Authentication
- Tailwind CSS

## Features

- Admin dashboard
- Employee/Cashier POS terminal
- Kitchen Display System
- Floor and table management
- Product and category management
- Customer management
- Coupons and payments
- Real-time order status updates

## Environment Setup

Create `backend/.env` using `backend/.env.example`.

Create `frontend/.env` using `frontend/.env.example`.

Do not commit real `.env` files.
